## 📌 Projetos em Destaque

### 👤 BEA-System

Projeto de conclusão de curso voltado para automatização do controle de presença acadêmica utilizando tecnologia BLE, dispositivos Beacon e receptores ESP32.

**Principais funcionalidades**

* Registro automático de presença
* Monitoramento em tempo real da ocupação da instituição
* Relatórios diários e mensais
* Integração entre hardware e software

**Tecnologias**

`Java` `Spring Boot` `PostgreSQL` `JavaScript` `ESP32` `BLE`
